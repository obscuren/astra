# Ruins POI V2 — Wall Network Megastructures Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Generate ancient megastructure ruins as continuous wall networks spanning detail tiles, driven by BSP subdivision with noise decay, per-civilization visual styles, and multi-tile overworld continuity.

**Architecture:** BSP recursively subdivides the ruin footprint with variable depth — shallow splits produce thick monumental walls, deep splits create dense room clusters around 2–4 nucleus points. After wall placement, flood-fill identifies rooms for themed content. Gradient + sectoral decay erodes walls, then composable post-processing stamps add character (battle-scarred, infested, etc.). Edge continuity uses shared-seed wall stubs at ruin-to-ruin tile boundaries. A new `CivConfig` struct extends `CivStyle` with glyph/color palettes per civilization.

**Tech Stack:** C++20, terminal renderer, existing noise.h (fbm), existing TileMap/FixtureData system, existing PlacementScorer.

```cpp
#include "astra/bsp_generator.h"
#include "astra/noise.h"

#include <algorithm>
#include <cmath>

namespace astra {

namespace {

// Distance from point to nearest nucleus
float dist_to_nearest(int x, int y,
                      const std::vector<std::pair<int,int>>& nuclei) {
    float min_d = 1e9f;
    for (auto& [nx, ny] : nuclei) {
        float d = std::sqrt(static_cast<float>((x-nx)*(x-nx) + (y-ny)*(y-ny)));
        min_d = std::min(min_d, d);
    }
    return min_d;
}

} // namespace

int BspGenerator::nucleus_depth(int base_depth, int x, int y,
                                const std::vector<std::pair<int,int>>& nuclei,
                                int radius) const {
    float d = dist_to_nearest(x, y, nuclei);
    if (d < radius * 0.3f) return base_depth + 3;
    if (d < radius * 0.6f) return base_depth + 2;
    if (d < radius)         return base_depth + 1;
    return base_depth;
}

void BspGenerator::subdivide(std::vector<BspNode>& nodes, int node_idx,
                              int max_depth,
                              const std::vector<std::pair<int,int>>& nuclei,
                              float regularity, std::mt19937& rng) const {
    auto& node = nodes[node_idx];
    int cx = node.area.x + node.area.w / 2;
    int cy = node.area.y + node.area.h / 2;

    // Determine local max depth based on nucleus proximity
    int local_max = nucleus_depth(max_depth, cx, cy, nuclei,
                                  std::max(node.area.w, node.area.h));

    if (node.depth >= local_max) {
        node.is_leaf = true;
        // Check if this leaf is near a nucleus
        float d = dist_to_nearest(cx, cy, nuclei);
        if (d < 20.0f) node.is_nucleus = true;
        return;
    }

    // Minimum partition size: outer walls need room, inner walls can be smaller
    int min_size = (node.depth < 2) ? 20 : 8;
    if (node.area.w < min_size * 2 && node.area.h < min_size * 2) {
        node.is_leaf = true;
        float d = dist_to_nearest(cx, cy, nuclei);
        if (d < 20.0f) node.is_nucleus = true;
        return;
    }

    // Choose split direction: prefer splitting the longer axis
    // regularity biases toward center split, low regularity = more random
    bool split_horizontal;
    if (node.area.w > node.area.h * 1.3f) {
        split_horizontal = false; // split vertically (along width)
    } else if (node.area.h > node.area.w * 1.3f) {
        split_horizontal = true;
    } else {
        split_horizontal = std::uniform_int_distribution<int>(0, 1)(rng);
    }

    // Split position: blend between center and random
    std::uniform_real_distribution<float> ratio_dist(0.35f, 0.65f);
    float random_ratio = ratio_dist(rng);
    float center_ratio = 0.5f;
    float split_ratio = regularity * center_ratio + (1.0f - regularity) * random_ratio;

    BspNode child_a, child_b;
    child_a.depth = node.depth + 1;
    child_b.depth = node.depth + 1;

    if (split_horizontal) {
        int split_y = node.area.y + static_cast<int>(node.area.h * split_ratio);
        // Clamp to ensure minimum sizes
        split_y = std::clamp(split_y,
                             node.area.y + min_size,
                             node.area.y + node.area.h - min_size);
        child_a.area = {node.area.x, node.area.y,
                        node.area.w, split_y - node.area.y};
        child_b.area = {node.area.x, split_y,
                        node.area.w, node.area.y + node.area.h - split_y};
    } else {
        int split_x = node.area.x + static_cast<int>(node.area.w * split_ratio);
        split_x = std::clamp(split_x,
                             node.area.x + min_size,
                             node.area.x + node.area.w - min_size);
        child_a.area = {node.area.x, node.area.y,
                        split_x - node.area.x, node.area.h};
        child_b.area = {split_x, node.area.y,
                        node.area.x + node.area.w - split_x, node.area.h};
    }

    int idx_a = static_cast<int>(nodes.size());
    nodes.push_back(child_a);
    int idx_b = static_cast<int>(nodes.size());
    nodes.push_back(child_b);

    // Update parent AFTER pushing (indices are stable since we don't erase)
    nodes[node_idx].child_a = idx_a;
    nodes[node_idx].child_b = idx_b;

    subdivide(nodes, idx_a, max_depth, nuclei, regularity, rng);
    subdivide(nodes, idx_b, max_depth, nuclei, regularity, rng);
}

void BspGenerator::materialize_walls(TileMap& map,
                                     const RuinPlan& plan) const {
    // For each non-leaf node, draw walls along its split boundary.
    // Wall thickness decreases with depth.
    unsigned noise_seed = static_cast<unsigned>(
        plan.footprint.x * 7919 + plan.footprint.y * 104729);

    for (const auto& node : plan.bsp_nodes) {
        if (node.is_leaf) continue;
        if (node.child_a < 0 || node.child_b < 0) continue;

        const auto& a = plan.bsp_nodes[node.child_a];
        const auto& b = plan.bsp_nodes[node.child_b];

        // Determine wall thickness based on depth + civ bias
        float base_thick = 0.0f;
        if (node.depth == 0)      base_thick = 5.0f;
        else if (node.depth == 1) base_thick = 4.0f;
        else if (node.depth == 2) base_thick = 3.0f;
        else if (node.depth == 3) base_thick = 2.0f;
        else                      base_thick = 1.0f;
        base_thick *= plan.civ.wall_thickness_bias;
        int thickness = std::max(1, static_cast<int>(base_thick));
        int half = thickness / 2;

        // Determine if horizontal or vertical split
        bool horizontal_split = (a.area.y != b.area.y);

        if (horizontal_split) {
            // Wall runs horizontally at the boundary between a and b
            int wall_y = a.area.y + a.area.h;
            int x_start = node.area.x;
            int x_end = node.area.x + node.area.w;

            for (int x = x_start; x < x_end; ++x) {
                // Noise-based thickness variation
                float n = fbm(static_cast<float>(x), static_cast<float>(wall_y),
                              noise_seed, 0.1f, 2);
                int local_half = half + static_cast<int>((n - 0.5f) * 2.0f);
                local_half = std::max(0, local_half);

                for (int dy = -local_half; dy <= local_half; ++dy) {
                    int wy = wall_y + dy;
                    if (wy < 0 || wy >= map.height()) continue;
                    if (x < 0 || x >= map.width()) continue;
                    Tile cur = map.get(x, wy);
                    // Don't overwrite water
                    if (cur == Tile::Water) continue;
                    map.set(x, wy, Tile::Wall);
                }
            }
        } else {
            // Wall runs vertically
            int wall_x = a.area.x + a.area.w;
            int y_start = node.area.y;
            int y_end = node.area.y + node.area.h;

            for (int y = y_start; y < y_end; ++y) {
                float n = fbm(static_cast<float>(wall_x), static_cast<float>(y),
                              noise_seed, 0.1f, 2);
                int local_half = half + static_cast<int>((n - 0.5f) * 2.0f);
                local_half = std::max(0, local_half);

                for (int dx = -local_half; dx <= local_half; ++dx) {
                    int wx = wall_x + dx;
                    if (wx < 0 || wx >= map.width()) continue;
                    if (y < 0 || y >= map.height()) continue;
                    Tile cur = map.get(wx, y);
                    if (cur == Tile::Water) continue;
                    map.set(wx, y, Tile::Wall);
                }
            }
        }
    }

    // Carve gaps/openings in walls to ensure navigability
    // For each non-leaf node, carve 1-3 openings in its wall
    std::mt19937 gap_rng(noise_seed + 42);
    for (const auto& node : plan.bsp_nodes) {
        if (node.is_leaf) continue;
        if (node.child_a < 0 || node.child_b < 0) continue;

        const auto& a = plan.bsp_nodes[node.child_a];
        const auto& b = plan.bsp_nodes[node.child_b];

        bool horizontal_split = (a.area.y != b.area.y);
        int gap_count = std::uniform_int_distribution<int>(1, 3)(gap_rng);
        int gap_width = (node.depth < 2) ? 3 : 2;

        for (int g = 0; g < gap_count; ++g) {
            if (horizontal_split) {
                int wall_y = a.area.y + a.area.h;
                int x_start = node.area.x + 2;
                int x_end = node.area.x + node.area.w - 2 - gap_width;
                if (x_end <= x_start) continue;
                int gx = std::uniform_int_distribution<int>(x_start, x_end)(gap_rng);

                for (int dx = 0; dx < gap_width; ++dx) {
                    int thickness_here = std::max(1, static_cast<int>(base_thick_for_depth(node.depth) * plan.civ.wall_thickness_bias));
                    int h = thickness_here / 2 + 1;
                    for (int dy = -h; dy <= h; ++dy) {
                        int wx = gx + dx;
                        int wy = wall_y + dy;
                        if (wx >= 0 && wx < map.width() && wy >= 0 && wy < map.height()) {
                            if (map.get(wx, wy) == Tile::Wall)
                                map.set(wx, wy, Tile::IndoorFloor);
                        }
                    }
                }
            } else {
                int wall_x = a.area.x + a.area.w;
                int y_start = node.area.y + 2;
                int y_end = node.area.y + node.area.h - 2 - gap_width;
                if (y_end <= y_start) continue;
                int gy = std::uniform_int_distribution<int>(y_start, y_end)(gap_rng);

                for (int dy = 0; dy < gap_width; ++dy) {
                    int thickness_here = std::max(1, static_cast<int>(base_thick_for_depth(node.depth) * plan.civ.wall_thickness_bias));
                    int h = thickness_here / 2 + 1;
                    for (int dx = -h; dx <= h; ++dx) {
                        int wx = wall_x + dx;
                        int wy = gy + dy;
                        if (wx >= 0 && wx < map.width() && wy >= 0 && wy < map.height()) {
                            if (map.get(wx, wy) == Tile::Wall)
                                map.set(wx, wy, Tile::IndoorFloor);
                        }
                    }
                }
            }
        }
    }
}

void BspGenerator::generate(TileMap& map, RuinPlan& plan,
                            std::mt19937& rng) const {
    // Select 2-4 nucleus points within the footprint
    int nucleus_count = std::uniform_int_distribution<int>(2, 4)(rng);
    std::vector<std::pair<int,int>> nuclei;
    std::uniform_int_distribution<int> nx_dist(
        plan.footprint.x + plan.footprint.w / 4,
        plan.footprint.x + plan.footprint.w * 3 / 4);
    std::uniform_int_distribution<int> ny_dist(
        plan.footprint.y + plan.footprint.h / 4,
        plan.footprint.y + plan.footprint.h * 3 / 4);

    for (int i = 0; i < nucleus_count; ++i) {
        nuclei.push_back({nx_dist(rng), ny_dist(rng)});
    }

    // Initialize BSP tree with root node
    plan.bsp_nodes.clear();
    BspNode root;
    root.area = plan.footprint;
    root.depth = 0;
    plan.bsp_nodes.push_back(root);

    // Base max depth for non-nucleus areas
    int base_max_depth = 3;

    subdivide(plan.bsp_nodes, 0, base_max_depth, nuclei,
              plan.civ.split_regularity, rng);

    // Place IndoorFloor in all leaf areas first
    for (const auto& node : plan.bsp_nodes) {
        if (!node.is_leaf) continue;
        for (int y = node.area.y; y < node.area.y + node.area.h; ++y) {
            for (int x = node.area.x; x < node.area.x + node.area.w; ++x) {
                if (x < 0 || x >= map.width() || y < 0 || y >= map.height())
                    continue;
                Tile cur = map.get(x, y);
                if (cur != Tile::Water) {
                    map.set(x, y, Tile::IndoorFloor);
                }
            }
        }
    }

    // Materialize walls along BSP split boundaries
    materialize_walls(map, plan);
}

namespace {

float base_thick_for_depth(int depth) {
    if (depth == 0) return 5.0f;
    if (depth == 1) return 4.0f;
    if (depth == 2) return 3.0f;
    if (depth == 3) return 2.0f;
    return 1.0f;
}

} // namespace

} // namespace astra
```

```cpp
namespace {

bool is_walkable(Tile t) {
    return t == Tile::Floor || t == Tile::IndoorFloor || t == Tile::Path;
}

int coord_key(int x, int y, int width) {
    return y * width + x;
}

} // namespace

BuildingType RoomIdentifier::theme_for_geometry(
        const RuinRoom& room, const CivConfig& civ,
        std::mt19937& rng) const {
    int area = static_cast<int>(room.floor_tiles.size());
    float aspect = static_cast<float>(room.bounds.w) /
                   std::max(1, room.bounds.h);

    // If this room is in a nucleus cluster, pick from civ preferences
    if (room.is_nucleus && !civ.preferred_rooms.empty()) {
        int idx = std::uniform_int_distribution<int>(
            0, static_cast<int>(civ.preferred_rooms.size()) - 1)(rng);
        return civ.preferred_rooms[idx];
    }

    // Organic assignment by geometry
    if (area > 300)                         return BuildingType::GreatHall;
    if (area < 80)                          return BuildingType::Vault;
    if (aspect > 2.5f || aspect < 0.4f)     return BuildingType::Archive; // elongated
    if (area > 150)                         return BuildingType::Temple;
    return BuildingType::Observatory;
}

void RoomIdentifier::identify(const TileMap& map, RuinPlan& plan,
                              std::mt19937& rng) const {
    const auto& fp = plan.footprint;
    int w = map.width();

    // Track visited tiles
    std::vector<bool> visited(w * map.height(), false);

    // Mark all wall tiles as visited (not part of any room)
    for (int y = fp.y; y < fp.y + fp.h; ++y) {
        for (int x = fp.x; x < fp.x + fp.w; ++x) {
            if (x < 0 || x >= w || y < 0 || y >= map.height()) continue;
            Tile t = map.get(x, y);
            if (!is_walkable(t)) {
                visited[coord_key(x, y, w)] = true;
            }
        }
    }

    // Flood-fill to discover rooms
    plan.rooms.clear();
    static constexpr int dx[] = {0, 0, -1, 1};
    static constexpr int dy[] = {-1, 1, 0, 0};

    for (int y = fp.y; y < fp.y + fp.h; ++y) {
        for (int x = fp.x; x < fp.x + fp.w; ++x) {
            int key = coord_key(x, y, w);
            if (visited[key]) continue;
            visited[key] = true;

            // BFS flood-fill
            RuinRoom room;
            room.floor_tiles.push_back({x, y});
            int min_x = x, max_x = x, min_y = y, max_y = y;

            std::queue<std::pair<int,int>> q;
            q.push({x, y});

            while (!q.empty()) {
                auto [cx, cy] = q.front();
                q.pop();

                for (int d = 0; d < 4; ++d) {
                    int nx = cx + dx[d];
                    int ny = cy + dy[d];

                    // Stay within footprint
                    if (nx < fp.x || nx >= fp.x + fp.w ||
                        ny < fp.y || ny >= fp.y + fp.h)
                        continue;

                    int nkey = coord_key(nx, ny, w);
                    if (visited[nkey]) continue;
                    visited[nkey] = true;

                    if (!is_walkable(map.get(nx, ny))) continue;

                    room.floor_tiles.push_back({nx, ny});
                    min_x = std::min(min_x, nx);
                    max_x = std::max(max_x, nx);
                    min_y = std::min(min_y, ny);
                    max_y = std::max(max_y, ny);
                    q.push({nx, ny});
                }
            }

            // Minimum room size: 8x8 area = 64 tiles
            if (static_cast<int>(room.floor_tiles.size()) < 64) continue;

            room.bounds = {min_x, min_y, max_x - min_x + 1, max_y - min_y + 1};

            // Check if room overlaps with a nucleus BSP leaf
            int room_cx = min_x + (max_x - min_x) / 2;
            int room_cy = min_y + (max_y - min_y) / 2;
            for (const auto& node : plan.bsp_nodes) {
                if (!node.is_nucleus) continue;
                if (node.area.contains(room_cx, room_cy)) {
                    room.is_nucleus = true;
                    break;
                }
            }

            room.theme = theme_for_geometry(room, plan.civ, rng);
            plan.rooms.push_back(std::move(room));
        }
    }
}

} // namespace astra
```

```cpp
#include "astra/ruin_types.h"
#include "astra/noise.h"

#include <cmath>

namespace astra {

namespace {

bool is_wall(Tile t) {
    return t == Tile::Wall || t == Tile::StructuralWall;
}

bool is_floor(Tile t) {
    return t == Tile::Floor || t == Tile::IndoorFloor || t == Tile::Path;
}

void stamp_battle_scarred(TileMap& map, const Rect& fp,
                          float intensity, std::mt19937& rng) {
    // Place 2-5 blast craters
    std::uniform_real_distribution<float> chance(0.0f, 1.0f);
    int crater_count = 2 + static_cast<int>(intensity * 3.0f);
    std::uniform_int_distribution<int> cx_dist(fp.x + 5, fp.x + fp.w - 5);
    std::uniform_int_distribution<int> cy_dist(fp.y + 5, fp.y + fp.h - 5);

    for (int c = 0; c < crater_count; ++c) {
        int cx = cx_dist(rng);
        int cy = cy_dist(rng);
        int radius = 3 + static_cast<int>(intensity * 4.0f);

        for (int dy = -radius; dy <= radius; ++dy) {
            for (int dx = -radius; dx <= radius; ++dx) {
                int dist2 = dx * dx + dy * dy;
                if (dist2 > radius * radius) continue;
                int x = cx + dx;
                int y = cy + dy;
                if (x < 0 || x >= map.width() || y < 0 || y >= map.height())
                    continue;

                float dist_ratio = std::sqrt(static_cast<float>(dist2)) /
                                   static_cast<float>(radius);

                if (dist_ratio < 0.5f) {
                    // Inner crater: clear to floor + debris
                    if (is_wall(map.get(x, y))) {
                        map.set(x, y, Tile::Floor);
                    }
                    if (map.fixture_id(x, y) < 0 && chance(rng) < 0.4f) {
                        map.add_fixture(x, y, make_fixture(FixtureType::Debris));
                    }
                } else if (dist_ratio < 0.8f) {
                    // Outer ring: scatter debris on floors
                    if (is_floor(map.get(x, y)) && map.fixture_id(x, y) < 0) {
                        if (chance(rng) < 0.3f) {
                            map.add_fixture(x, y, make_fixture(FixtureType::Debris));
                        }
                    }
                }
                // Scorch tint on surviving walls in crater area
                if (is_wall(map.get(x, y))) {
                    map.set_custom_flag(x, y, 0x02); // CF_RUIN_TINT
                }
            }
        }
    }
}

void stamp_infested(TileMap& map, const Rect& fp,
                    float intensity, std::mt19937& rng) {
    // Scatter organic fixtures in rooms
    std::uniform_real_distribution<float> chance(0.0f, 1.0f);
    float density = 0.05f + intensity * 0.10f;

    for (int y = fp.y; y < fp.y + fp.h; ++y) {
        for (int x = fp.x; x < fp.x + fp.w; ++x) {
            if (x < 0 || x >= map.width() || y < 0 || y >= map.height())
                continue;
            if (!is_floor(map.get(x, y))) continue;
            if (map.fixture_id(x, y) >= 0) continue;

            if (chance(rng) < density) {
                // Organic growth: use FloraMushroom or FloraLichen
                FixtureType type = (chance(rng) < 0.6f)
                    ? FixtureType::FloraMushroom
                    : FixtureType::FloraLichen;
                map.add_fixture(x, y, make_fixture(type));
            }
        }
    }
}

void stamp_flooded(TileMap& map, const Rect& fp,
                   float intensity, std::mt19937& rng) {
    // Flood low areas with water using noise
    unsigned seed = static_cast<unsigned>(fp.x * 3571 + fp.y * 9973);
    float threshold = 0.55f - intensity * 0.15f;  // lower = more flooding

    for (int y = fp.y; y < fp.y + fp.h; ++y) {
        for (int x = fp.x; x < fp.x + fp.w; ++x) {
            if (x < 0 || x >= map.width() || y < 0 || y >= map.height())
                continue;
            if (!is_floor(map.get(x, y))) continue;

            float n = fbm(static_cast<float>(x), static_cast<float>(y),
                          seed, 0.05f, 3);
            if (n < threshold) {
                // Remove fixture if present
                if (map.fixture_id(x, y) >= 0) {
                    map.remove_fixture(x, y);
                }
                map.set(x, y, Tile::Water);
            }
        }
    }
}

void stamp_excavated(TileMap& map, const Rect& fp,
                     float intensity, std::mt19937& rng) {
    // Clean rectangular sections carved out by scavengers
    int section_count = 1 + static_cast<int>(intensity * 2.0f);
    std::uniform_int_distribution<int> sx_dist(fp.x, fp.x + fp.w - 15);
    std::uniform_int_distribution<int> sy_dist(fp.y, fp.y + fp.h - 10);
    std::uniform_int_distribution<int> sw_dist(8, 15);
    std::uniform_int_distribution<int> sh_dist(6, 10);

    for (int s = 0; s < section_count; ++s) {
        int sx = sx_dist(rng);
        int sy = sy_dist(rng);
        int sw = sw_dist(rng);
        int sh = sh_dist(rng);

        for (int y = sy; y < sy + sh && y < fp.y + fp.h; ++y) {
            for (int x = sx; x < sx + sw && x < fp.x + fp.w; ++x) {
                if (x < 0 || x >= map.width() || y < 0 || y >= map.height())
                    continue;
                // Clean cut: walls become floor, fixtures removed
                if (is_wall(map.get(x, y))) {
                    map.set(x, y, Tile::Floor);
                }
                if (map.fixture_id(x, y) >= 0) {
                    map.remove_fixture(x, y);
                }
            }
        }
    }
}

} // namespace

void apply_ruin_stamps(TileMap& map, const RuinPlan& plan,
                       Biome biome, std::mt19937& rng) {
    for (const auto& stamp : plan.stamps) {
        switch (stamp.type) {
            case RuinStampType::BattleScarred:
                stamp_battle_scarred(map, plan.footprint, stamp.intensity, rng);
                break;
            case RuinStampType::Infested:
                stamp_infested(map, plan.footprint, stamp.intensity, rng);
                break;
            case RuinStampType::Flooded:
                stamp_flooded(map, plan.footprint, stamp.intensity, rng);
                break;
            case RuinStampType::Excavated:
                stamp_excavated(map, plan.footprint, stamp.intensity, rng);
                break;
        }
    }
}

} // namespace astra
```

```cpp
// Select CivConfig based on props or explicit name
CivConfig select_civ(const MapProperties& props, std::mt19937& rng,
                     const std::string& civ_name) {
    if (!civ_name.empty()) {
        return civ_config_by_name(civ_name);
    }

    // Use lore architecture if available
    if (props.lore_primary_civ_index >= 0) {
        return civ_config_for_architecture(props.lore_alien_architecture);
    }

    // Random selection
    int pick = std::uniform_int_distribution<int>(0, 3)(rng);
    switch (pick) {
        case 0:  return civ_config_monolithic();
        case 1:  return civ_config_baroque();
        case 2:  return civ_config_crystal();
        default: return civ_config_industrial();
    }
}

// Determine which stamps to apply based on map properties
std::vector<RuinStampConfig> select_stamps(const MapProperties& props,
                                           std::mt19937& rng) {
    std::vector<RuinStampConfig> stamps;
    std::uniform_real_distribution<float> chance(0.0f, 1.0f);

    if (props.lore_battle_site) {
        stamps.push_back({RuinStampType::BattleScarred, 0.5f + chance(rng) * 0.3f});
    }

    // Random chance of infestation on non-frozen biomes
    if (props.biome != Biome::Ice && props.biome != Biome::Volcanic) {
        if (chance(rng) < 0.3f) {
            stamps.push_back({RuinStampType::Infested, 0.3f + chance(rng) * 0.4f});
        }
    }

    // Chance of partial flooding on wet biomes
    if (props.biome == Biome::Marsh || props.biome == Biome::Jungle ||
        props.biome == Biome::Forest) {
        if (chance(rng) < 0.25f) {
            stamps.push_back({RuinStampType::Flooded, 0.3f + chance(rng) * 0.3f});
        }
    }

    return stamps;
}

void RuinGenerator::place_room_furniture(TileMap& map, const RuinRoom& room,
                                         const CivConfig& civ,
                                         std::mt19937& rng) const {
    // Get furniture palette for this room's theme
    // Reuse existing palette system with a ruin CivStyle
    CivStyle ruin_style;
    ruin_style.name = civ.name;
    ruin_style.wall_tile = Tile::Wall;
    ruin_style.floor_tile = Tile::IndoorFloor;
    ruin_style.lighting = FixtureType::Torch;
    ruin_style.storage = FixtureType::Crate;
    ruin_style.seating = FixtureType::Debris;
    ruin_style.cooking = FixtureType::Debris;
    ruin_style.knowledge = FixtureType::Debris;
    ruin_style.display = FixtureType::Debris;
    ruin_style.decay = 0.5f;

    auto palette = furniture_palette(room.theme, ruin_style);

    // Simple placement: scatter furniture on floor tiles
    std::uniform_real_distribution<float> chance(0.0f, 1.0f);
    std::uniform_int_distribution<int> tile_pick(
        0, std::max(0, static_cast<int>(room.floor_tiles.size()) - 1));

    for (const auto& group : palette.groups) {
        if (chance(rng) > group.frequency) continue;

        int count = std::uniform_int_distribution<int>(
            group.min_count, group.max_count)(rng);

        for (int i = 0; i < count; ++i) {
            // Try a few random positions
            for (int attempt = 0; attempt < 10; ++attempt) {
                int idx = tile_pick(rng);
                auto [fx, fy] = room.floor_tiles[idx];
                if (map.fixture_id(fx, fy) >= 0) continue;
                if (map.get(fx, fy) != Tile::IndoorFloor) continue;

                map.add_fixture(fx, fy, make_fixture(group.primary));
                break;
            }
        }
    }
}

void RuinGenerator::apply_edge_continuity(TileMap& map,
                                          const RuinPlan& plan,
                                          const MapProperties& props) const {
    // For each connected edge, generate wall stubs from shared seed
    auto edge_seed = [&](int ax, int ay, int bx, int by) -> unsigned {
        int min_x = std::min(ax, bx), min_y = std::min(ay, by);
        int max_x = std::max(ax, bx), max_y = std::max(ay, by);
        return static_cast<unsigned>(min_x * 7919 + min_y * 104729 +
                                     max_x * 15487 + max_y * 32771);
    };

    auto place_edge_walls = [&](int edge_start, int edge_end,
                                bool is_vertical, int fixed_coord,
                                unsigned seed) {
        // Generate 3-6 wall stub positions along this edge
        std::mt19937 edge_rng(seed);
        int stub_count = std::uniform_int_distribution<int>(3, 6)(edge_rng);
        int range = edge_end - edge_start;
        if (range < 10) return;

        for (int s = 0; s < stub_count; ++s) {
            int pos = edge_start + std::uniform_int_distribution<int>(
                2, range - 2)(edge_rng);
            int stub_len = std::uniform_int_distribution<int>(3, 8)(edge_rng);
            int thickness = std::uniform_int_distribution<int>(2, 4)(edge_rng);

            for (int i = 0; i < stub_len; ++i) {
                for (int t = 0; t < thickness; ++t) {
                    int x, y;
                    if (is_vertical) {
                        x = fixed_coord + (fixed_coord == 0 ? i : -i);
                        y = pos + t - thickness / 2;
                    } else {
                        x = pos + t - thickness / 2;
                        y = fixed_coord + (fixed_coord == 0 ? i : -i);
                    }
                    if (x >= 0 && x < map.width() && y >= 0 && y < map.height()) {
                        if (map.get(x, y) != Tile::Water) {
                            map.set(x, y, Tile::Wall);
                            map.set_custom_flag(x, y, 0x02); // CF_RUIN_TINT
                        }
                    }
                }
            }
        }
    };

    int ow_x = props.overworld_x;
    int ow_y = props.overworld_y;

    if (plan.edge_n) {
        unsigned seed = edge_seed(ow_x, ow_y, ow_x, ow_y - 1);
        place_edge_walls(plan.footprint.x, plan.footprint.x + plan.footprint.w,
                         false, 0, seed);
    }
    if (plan.edge_s) {
        unsigned seed = edge_seed(ow_x, ow_y, ow_x, ow_y + 1);
        place_edge_walls(plan.footprint.x, plan.footprint.x + plan.footprint.w,
                         false, map.height() - 1, seed);
    }
    if (plan.edge_w) {
        unsigned seed = edge_seed(ow_x, ow_y, ow_x - 1, ow_y);
        place_edge_walls(plan.footprint.y, plan.footprint.y + plan.footprint.h,
                         true, 0, seed);
    }
    if (plan.edge_e) {
        unsigned seed = edge_seed(ow_x, ow_y, ow_x + 1, ow_y);
        place_edge_walls(plan.footprint.y, plan.footprint.y + plan.footprint.h,
                         true, map.width() - 1, seed);
    }
}

Rect RuinGenerator::generate(TileMap& map, const TerrainChannels& channels,
                             const MapProperties& props, std::mt19937& rng,
                             const std::string& civ_name) const {
    // 1. Select civilization
    CivConfig civ = select_civ(props, rng, civ_name);

    // 2. Determine footprint (80-95% of map)
    int margin_x = map.width() / 20;    // 5% margin each side
    int margin_y = map.height() / 20;
    int foot_w = map.width() - margin_x * 2;
    int foot_h = map.height() - margin_y * 2;

    // 3. Placement scoring
    PlacementScorer scorer;
    auto placement = scorer.score(channels, map, foot_w, foot_h, margin_x);
    if (!placement.valid) return {};

    // 4. Build ruin plan
    RuinPlan plan;
    plan.footprint = placement.footprint;
    plan.civ = civ;
    plan.stamps = select_stamps(props, rng);
    plan.base_decay = 0.5f;

    // Determine connected edges from neighbor tiles
    plan.edge_n = (props.detail_neighbor_n == Tile::OW_Ruins);
    plan.edge_s = (props.detail_neighbor_s == Tile::OW_Ruins);
    plan.edge_e = (props.detail_neighbor_e == Tile::OW_Ruins);
    plan.edge_w = (props.detail_neighbor_w == Tile::OW_Ruins);

    // 5. BSP subdivision + wall materialization
    BspGenerator bsp;
    bsp.generate(map, plan, rng);

    // 6. Room identification + theme assignment
    RoomIdentifier room_id;
    room_id.identify(map, plan, rng);

    // 7. Furniture placement per room
    for (const auto& room : plan.rooms) {
        place_room_furniture(map, room, civ, rng);
    }

    // 8. Decay (gradient + sectoral)
    DecayContext decay_ctx;
    decay_ctx.age_decay = plan.base_decay;
    decay_ctx.use_gradient = true;
    decay_ctx.gradient_footprint = plan.footprint;
    decay_ctx.use_sectoral = true;
    decay_ctx.sectoral_variance = 0.3f;

    RuinDecay decay;
    decay.apply(map, plan.footprint, decay_ctx, props.biome, rng);

    // 9. Post-processing stamps
    apply_ruin_stamps(map, plan, props.biome, rng);

    // 10. Edge continuity
    apply_edge_continuity(map, plan, props);

    return plan.footprint;
}
```

### What This Plan Does NOT Cover (Future Work)

- **Overworld rendering with per-civ glyphs** — the overworld currently uses SG_Pi/SG_Omega for ruins. Per-civ glyph rendering (░██▓ vs ╔═╗║) needs changes to the overworld theme system and is a separate task.
- **New irregular ruin stamp shapes on overworld** — current stamps are 1x1, 2x2 L-shape, 3x3. More varied irregular shapes are a separate addition to `overworld_stamps.cpp`.
- **Overworld ruin cluster generation** — current placement uses existing `ruin_stamps[]` and `place_n()`. The configurable frequency / scaled-by-significance system needs overworld generator changes.
- **NPC spawning in ruins** — scavengers, guardians, creatures in specific room types.
- **Loot tables per room theme** — vaults have better loot, archives have data terminals.
- **Per-civ wall tint colors** — currently `CF_RUIN_TINT` produces mossy green or crumbling brown. Extending to use `CivConfig::color_tint` requires `terminal_theme.cpp` changes.

These items build on this plan's foundation and should be planned separately once the core wall network generation is working.
